## What I do

- Validate design tokens and color schemes
- Ensure consistent spacing, typography, and icon usage
- Check Flutter widget consistency (buttons, inputs, cards)
- Suggest theming improvements using ThemeData
- Verify accessibility standards (contrast ratios, screen reader support)
- Review responsive design implementation
